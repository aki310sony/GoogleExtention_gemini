# CSS Element Rotator Chrome Extension

これは、表示しているWebページ上の任意の要素を、CSSセレクターを使って指定し、CSSアニメーションで回転させることができるChrome拡張機能です。

## ファイル構成と役割

この拡張機能は、以下のファイルで構成されています。

*   `manifest.json`
    *   **役割**: 拡張機能の設計図（マニフェスト）です。
    *   **詳細**: 拡張機能の名前、バージョン、説明、必要な権限（アクティブなタブへのアクセス許可など）を定義します。また、ポップアップウィンドウやツールバーに表示するアイコンのファイルパスもここで指定します。Chromeはこのファイルを最初に読み込み、拡張機能の全体像を把握します。

*   `popup.html`
    *   **役割**: ポップアップウィンドウのUI（ユーザーインターフェース）を定義するHTMLファイルです。
    *   **詳細**: 拡張機能のアイコンをクリックしたときに表示される小さなウィンドウの骨格を定義します。CSSセレクターを入力するためのテキストボックスや、アニメーションを開始するためのボタンが配置されています。

*   `popup.css`
    *   **役割**: `popup.html`の見た目を整えるためのスタイルシートです。
    *   **詳細**: ポップアップウィンドウのレイアウト、フォント、色、ボタンのデザインなどを設定し、ユーザーが直感的に使えるように見た目を整えます。

*   `popup.js`
    *   **役割**: ポップアップウィンドウの動作を制御するJavaScriptファイルです。
    *   **詳細**: 「Rotate!」ボタンがクリックされたことを検知し、入力ボックスからCSSセレクターの値を取得します。そして、そのセレクター情報を現在開いているWebページ上で動作している`content.js`へ送信する橋渡しの役割を担います。

*   `content.js`
    *   **役割**: 実際にWebページ上でDOM操作を行い、アニメーションを適用するスクリプトです。
    *   **詳細**: `popup.js`からメッセージ（CSSセレクター）を受け取ります。受け取ったセレクターを元に`document.querySelector()`を使ってページ内の要素を検索し、見つかった要素のスタイルにCSSの回転アニメーションを適用します。ページ上で実際に変更を加えるのはこのファイルです。

*   `images/`
    *   **役割**: 拡張機能のアイコンファイルを格納するフォルダです。
    *   **詳細**: `manifest.json`で指定された、様々なサイズ（16x16, 48x48, 128x128）のアイコンが保存されています。これらのアイコンはChromeのツールバーや拡張機能管理ページに表示されます。
